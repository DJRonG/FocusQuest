## ğŸš€ Getting Started

Follow these steps to get a local copy up and running.

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or later)
- An active Google Gemini API key. You can get one from [Google AI Studio](https://aistudio.google.com/).

### Installation & Setup

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/your-username/focus-quest.git
    cd focus-quest
    ```

2.  **Install dependencies:**
    ```sh
    npm install
    ```

3.  **Set up environment variables:**
    The application requires your Gemini API key. Create a file named `.env.local` in the root and add your key:

    ```.env.local
    VITE_API_KEY=YOUR_GEMINI_API_KEY
    ```

### Running the Application

```sh
npm run dev
```

The application will be available at `http://localhost:5173`.

## ğŸš¢ Deployment

2.  **Host the static files:**
    Deploy the `dist` folder to services like Vercel, Netlify, or Firebase Hosting. Ensure `VITE_API_KEY` is configured in your provider's environment variables.
